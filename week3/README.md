# Week 3 – Functions and Lists

## About
This folder contains my third week of Python practice.  
Here I worked on defining functions, using default arguments, and exploring lists with indexing, slicing, methods, and sorting.

## Files
- **thirdLesson.py** – Lists in Python  
  Covers list creation, indexing, slicing, concatenation, repetition, and common list methods (`append`, `clear`, `copy`, `count`, `extend`, `insert`, `pop`, `remove`, `reverse`, `sort`).  
  Includes examples of shallow vs deep copy, iteration with `for` and `enumerate`, and custom sorting functions.

- **exercise01n02.py** – Functions with arguments  
  - `fun1(a, b)` – adds two numbers and returns the result.  
  - `fun2(a, b=5, c=10)` – adds three numbers, with default values for `b` and `c`.  
  Demonstrates how to use functions with and without default arguments.

## Topics Covered
- Function definitions and return values  
- Default arguments in functions  
- Lists: indexing, slicing, concatenation, repetition  
- List methods (`append`, `remove`, `sort`, etc.)  
- Shallow vs deep copy  
- Iterating over lists with `for` and `enumerate`  
- Custom sorting with functions
