# Week 2 – Strings and More Exercises

## About
This folder contains my second week of Python practice.  
Here I focused on working with strings, slicing, string functions, and solving small exercises with loops and conditions.

## Files
- **secondLesson.py** – Strings in Python  
  Covers string creation, indexing, slicing, concatenation, repetition, and common string methods (`upper`, `lower`, `strip`, `replace`, `count`, `find`, etc.). Also includes checks like `isalnum`, `isalpha`, `isdecimal`, `isupper`, `islower`.

- **exercise01.py** – Print triangle pattern  
  Reads a number `n` and prints a growing triangle of `*` characters.

- **exercise02.py** – Print triangle up and down  
  Similar to exercise01, but prints a triangle that grows up to `n` and then shrinks back down.

- **exercise03.py** – Diamond pattern  
  For `n=3`, prints a diamond shape made of `*` characters using spacing and loops.

- **exercise04.py** – Password validation  
  Program that checks if a password is valid. Conditions: at least 6 characters, one uppercase, one lowercase, one digit, and one special character.

## Topics Covered
- String basics (creation, indexing, slicing)  
- String operations (concatenation, repetition, replacement)  
- String methods (`upper`, `lower`, `strip`, `replace`, `count`, `find`, etc.)  
- Pattern printing with loops  
- Password validation with conditions and string checks
