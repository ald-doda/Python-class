# Exercises – Python Practice

## About
This folder contains my Python practice exercises.  
They are organized into subfolders by topic: basics, if-then, and if-then-else.  
Each section includes small programs that demonstrate fundamental programming concepts.

## Structure
- **basics/**  
  Introductory exercises covering variables, arithmetic operations, type conversions, and simple input/output.

- **if-then exercises/**  
  Tasks that use basic `if` statements to handle simple conditions.

- **if-then-else exercises/**  
  More advanced conditional tasks using `if-elif-else` structures for multiple cases.

## Topics Covered
- Basics of Python (variables, operators, input/output)  
- Conditional statements (`if`, `if-else`, `if-elif-else`)  
- Arithmetic operations and type conversions  
- Problem-solving through small exercises
