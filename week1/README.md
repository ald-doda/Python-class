# Week 1 – Python Basics

## About
This folder contains my first week of Python practice.  
Here I started with the very basics: variables, operators, type conversions, input/output, conditional statements, and loops.

## Files
- **firstLesson.py** – Introduction to Python  
  Covers variable initialization, arithmetic and logical operators, type conversions, binary/hex/octal numbers, comparison operators, user input, if-elif-else structures, and basic loops (while/for).

- **exercise01.py** – Temperature conversion  
  Program that reads temperature in Celsius and converts it to Fahrenheit using the formula `F = C * 9/5 + 32`.

- **exercise02.py** – Loan interest calculation  
  Program that reads loan amount and interest rate, calculates interest, and shows the total amount after a given number of years.

- **exercise02b.py** – Loan interest calculation (fixed values)  
  Same as exercise02, but values are defined inside the program instead of being entered by the user. Uses a loop to calculate the total after several years.

## Topics Covered
- Variables and data types  
- Arithmetic and logical operators  
- Type conversion (int, float, string)  
- Binary, hexadecimal, and octal numbers  
- Comparison operators  
- User input and conversion  
- Conditional statements (if-elif-else)  
- Loops (while, for) with break and continue
